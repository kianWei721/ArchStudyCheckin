# ArchStudyCheckin 产品需求文档（PRD）

## 1. 项目概述

### 1.1 项目名称
ArchStudyCheckin（系统架构设计师备考打卡 App）

### 1.2 项目定位
面向系统架构设计师考试备考人群的学习打卡工具，支持个人学习计划管理、每日打卡记录、学习提醒通知、小组监督互助等功能。

### 1.3 目标用户
- 正在备考系统架构设计师（高级）考试的 IT 从业者
- 有自律学习需求、希望通过打卡记录学习进度的用户
- 希望通过小组监督提高学习效率的用户

### 1.4 产品风格
- 简洁、专注、学习工具型
- 以功能为导向，减少干扰元素
- iOS 原生设计风格

## 4. 功能详细描述

### 4.1 F01 注册登录

**注册：**
- 输入：用户名、邮箱、密码
- 校验：用户名 2-20 字符，邮箱格式合法，密码 6-20 字符
- 成功后自动登录并返回 Token

**登录：**
- 输入：邮箱 + 密码
- 成功返回 JWT Token（有效期 7 天）
- Token 存储于 iOS Keychain

**退出登录：**
- 客户端清除本地 Token

### 4.2 F02 创建学习计划

- 输入：计划名称、科目（如"系统架构设计"）、目标天数、开始日期
- 每个用户可创建多个计划
- 计划状态：进行中 / 已完成 / 已放弃
- MVP 阶段不实现计划编辑和删除

### 4.3 F03 今日打卡

- 选择关联的学习计划
- 输入：学习时长（分钟）、学习内容摘要（可选，最多 200 字）
- 幂等设计：同一用户同一天同一计划只能打卡一次
- 如当日已打卡，重复提交则更新原记录（学习时长、内容）

### 4.4 F04 查看今日打卡状态

- 返回用户今日所有计划的打卡情况
- 已打卡显示打卡时间和学习内容
- 未打卡显示"待完成"

### 4.5 F05 查看月度打卡记录

- 输入：年月（如 2026-06）
- 返回该月每天的打卡状态（已打卡/未打卡）
- iOS 端以日历 UI 展示

### 4.6 F06 查看个人统计

- 累计打卡天数
- 当前连续打卡天数
- 总学习时长（分钟）
- 统计基于所有计划汇总

### 4.7 F07 创建学习小组

- 输入：小组名称、简介（可选）
- 自动生成 6 位邀请码（大写字母 + 数字）
- 创建者自动成为小组成员
- 每个用户最多创建 3 个小组

### 4.8 F08 邀请码加入小组

- 输入：6 位邀请码
- 校验邀请码有效性
- 加入成功后成为小组成员
- 每个用户最多加入 5 个小组

### 4.9 F09 查看小组今日打卡情况

- 显示小组所有成员的今日打卡状态
- 包含：成员昵称、是否已打卡、打卡时间

### 4.10 F10 本地学习提醒

- iOS 本地通知实现
- 用户可设置提醒时间（默认 20:00）
- 每日定时提醒"今日还未打卡，赶紧学习吧！"
- 如当日已打卡则不提醒（客户端本地判断）

## 6. 业务规则

### 6.1 打卡与通用规则

1. **打卡幂等**：同一用户 + 同一计划 + 同一天，只允许一条打卡记录。重复提交时更新原记录（学习时长、内容摘要），不新增记录。
2. **打卡日期规则**：打卡日期以用户本地时区当天日期为准，客户端提交时携带本地日期（格式 yyyy-MM-dd）。
3. **连续打卡计算**：从今日向前追溯，连续有打卡记录的天数（任意计划打卡均算）。
4. **邀请码唯一**：邀请码全局唯一，生成时需校验。
5. **Token 过期**：过期后客户端跳转登录页，需重新登录。

### 6.2 小组权限规则

1. **查看权限**：只有小组成员可以查看小组打卡数据，非成员无权访问。
2. **解散权限**：只有组长（创建者）可以解散小组。
3. **退出小组**：成员可以主动退出小组，组长不可退出（需先解散）。
4. **防重复加入**：同一用户不能重复加入同一小组，重复请求返回"已是成员"。
5. **邀请码全局唯一**：系统生成时校验唯一性，冲突则重新生成。

### 6.3 用户数据安全边界

1. **身份来源**：所有个人数据接口的 userId 必须从 JWT Token 中解析，后端不接受前端传入的 userId 作为可信身份参数。
2. **越权防护**：后端必须校验当前用户是否为目标小组成员，防止越权查看其他小组数据。
3. **数据隔离**：用户只能查看和操作自己的打卡记录、学习计划。

## 9. 验收标准

1. 用户可通过邮箱注册并登录
2. 登录后可创建学习计划
3. 可对某个计划进行今日打卡，同一天重复打卡更新原记录
4. 可查看今日所有计划打卡状态
5. 可查看指定月份的打卡日历
6. 可查看个人统计数据
7. 可创建小组并获取邀请码
8. 可通过邀请码加入小组
9. 可查看小组成员今日打卡状态
10. iOS 端可设置本地提醒时间并收到通知
